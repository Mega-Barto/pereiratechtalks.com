<p>📣 ¡Un gran evento organizado por todas las comunidades de tecnología en Pereira nos espera! Tendremos unas charlas buenísimas que no te puedes perder:</p> <p>1. Next.js: Server-side rendering your React App by Andrés Rodriguez Escudero (<a href="https://twitter.com/arodrigueze_" class="linkified">https://twitter.com/arodrigueze_</a>)</p> <p>🔎 Next.js es una herramienta que te permite hacer una aplicación web con React renderizando desde el lado del Server. Veremos las ventajas que hay cuando renderizas desde el lado del servidor, versus las ventajas de redenrizar desde el lado del cliente (Browsers) y como Next.js puede convivir con estos dos tipos de ventajas.</p> <p>2. Introducción a Amazon Web Services (AWS) by Steven Pineda Cortes (<a href="https://twitter.com/Steven2308" class="linkified">https://twitter.com/Steven2308</a>)</p> <p>🔎 Haremos un recorrido por varios de los servicios más populares de esta plataforma, incluyendo casos de uso. Encontrarás formas de facilitar y potencializar tus proyectos. EC2, RDS, IAM, SNS, SQS y S3 ya no serán solo siglas que escuchaste alguna vez. Veremos también Lambda, Athena y DynamoDB.</p> <p>3. Patrones de arquitectura - Android by Zorayda Gutiérrez Montes (<a href="https://www.facebook.com/zoraidayadi.gutierrezmontes" class="linkified">https://www.facebook.com/zoraidayadi.gutierrezmontes</a>)</p> <p>🔎 Conozcamos cómo estructurar el código de nuestras aplicaciones móviles de la mejor manera. Explicación y diferencias entre los patrones de arquitectura Model View Presenter (MVP) y Model View ViewModel (MVVM)</p> <p>4. Desarrollo serverless de interfaces conversacionales en Amazon Alexa y Google Home by Sergio Alexander Florez Galeano (<a href="https://twitter.com/xergioalex" class="linkified">https://twitter.com/xergioalex</a>)</p> <p>🔎 La voz es la nueva interface de usuario que está causando una gran revolución en el area tecnológica, y según Gartner para el 2020 más del 30% de las sesiones de busqueda en la web se llevarán a cabo sin una pantalla usando interfaces multimodales como la voz.</p> <p>Por otro lado Serverless es un modelo de computación en la nube donde bajo demanda solo te cobra por el tiempo de computo consumido y tiene un gran campo de acción en el mundo del IoT y el desarrollo de interfaces conversacionales como servicio de backend.</p> <p>Organizan:<br/>🚩 Python Pereira: <a href="https://pypereira.co/" class="linkified">https://pypereira.co/</a><br/>🚩 Joint Developer: <a href="https://sirius.utp.edu.co/jointdeveloper/" class="linkified">https://sirius.utp.edu.co/jointdeveloper/</a><br/>🚩 PereiraJs: <a href="https://pereirajs.org/" class="linkified">https://pereirajs.org/</a><br/>🚩 Pereira Tech Talks: <a href="https://pereiratechtalks.com/" class="linkified">https://pereiratechtalks.com/</a></p> <p>---</p> <p>🚩 Patrocinadores de este meetup:</p> <p>🏢 Universidad Tecnológica de Pereira: Agradecemos a la UTP por su gran apoyo ayudandonos en la gestión de espacios necesarios para desarrollar las actividades.</p> <p>🏢 Rocka (<a href="https://rocka.co" class="linkified">https://rocka.co</a>): Empresa de tecnología enfocada en el desarrollo de productos digitales con altos componentes de innovación.
